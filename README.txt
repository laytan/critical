All you need for Critical CSS generation and non-critical CSS defering / preloading.

== Description ==

A plugin for Wordpress that lets you easily generate, add and manage Critical CSS.
It also has the option to preload CSS files so they are non render blocking.

== Installation ==

* Upload the folder `critical` to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
